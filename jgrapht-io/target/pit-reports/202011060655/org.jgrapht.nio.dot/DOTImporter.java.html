37

38 
 * The provided graph object, where the imported graph will be stored, must be able to support the
39

40

42 
 *
43 
 * <p>
44

45

46

47 
 * multiple times.
48 
 *
49 
 * <p>
50

51

52

53 
 * graph vertex given the vertex identifier read from file.
54 
 *
